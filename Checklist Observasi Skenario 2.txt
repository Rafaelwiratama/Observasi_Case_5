   Checklist: ✔
   Analisis: cwnd mengecil sebagai respon congestion control.

PERTANYAAN 
1. TCP mendeteksi hilangnya segmen melalui duplicate ACK atau timeout.

2. Perbedaan :
	- Fast retransmit : retransmission dipicu oleh minimal 3 duplicate ACK sehingga lebih cepat
	- Timeout retransmission : retransmission terjadi setelah menunggu timer habis sehingga lebih lambat

3. Ketika loss terdeteksi, TCP mengurangi congestion window (cwnd) sehingga laju pengiriman menurun.

4. 	- Lost segment dikirim ulang (retransmission)
	- Data tetap diterima lengkap dan berurutan
Sehingga TCP tetap menjamin reliability.
